# Freelance CRM [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

<img src="static/logo.png" alt="Логотип" width="80" align="left">

Простая и удобная система для фрилансеров: управление проектами, задачами и счетами + автоматическая отправка счетов клиентам в Telegram.

CRUD + PDF + Telegram-уведомления

## Основной функционал

- Полное управление проектами (создание, редактирование, удаление)
- Задачи внутри проектов (CRUD, статусы, дедлайны, отметка просрочки)
- Счета (CRUD, автогенерация номера INV-год-XXX, статусы: черновик/отправлен/оплачен/частично/просрочен)
- Генерация PDF-счёта (скачивание)
- Отправка PDF-счёта клиенту в Telegram (асинхронно через Celery)
- Ежедневные уведомления в Telegram о просроченных и срочных задачах (9:00 и 18:00)
- Пагинация задач и счетов с сохранением позиции скролла
- Адаптивный интерфейс на Tailwind CSS + HTMX для динамики
- Поддержка Telegram-бота для регистрации клиентов (опционально)

## Технологический стек

- Backend: Django 5.2+
- Frontend: Tailwind CSS 2.2 + HTMX 1.9
- Асинхронные задачи: Celery + Redis
- База данных: SQLite (по умолчанию) / PostgreSQL (рекомендуется)
- Уведомления и отправка файлов: Telegram Bot API
- Генерация PDF: WeasyPrint
- Управление задачами: taskipy
